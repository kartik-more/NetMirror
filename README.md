# Netflix Clone 🎬

A Netflix landing page clone built using HTML, CSS and JavaScript.

## Features
- Netflix-style landing page
- English ↔ Hindi language toggle
- FAQ accordion section
- Responsive layout
- Video and media sections

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone the repository.
2. Open `NETFLIX.html` in your browser.
